## printf สน๏ฟฝรท๏ฟฝ๏ฟฝ๏ฟฝ

1. ๏ฟฝ๏ฟฝ CubeMX ๏ฟฝะด๒ฟชถ๏ฟฝำฆ USART๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ USART1๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝฮช Asynchronous๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝศท๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝสต๏ฟฝ๏ฟฝำฒ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาป๏ฟฝยก๏ฟฝ
2. ๏ฟฝ๏ฟฝ๏ฟฝาช๏ฟฝ๏ฟฝ๏ฟฝะถฯป๏ฟฝ๏ฟฝฬจ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝึด๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝะถฯฟ๏ฟฝ๏ฟฝรฃ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝึป๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝอฃ๏ฟฝ๏ฟฝ๏ฟฝอจ UART ๏ฟฝ๏ฟฝสผ๏ฟฝ๏ฟฝาฒ๏ฟฝ๏ฟฝ๏ฟฝิก๏ฟฝ
3. ๏ฟฝ๏ฟฝ [Core/Src/main.c](../../Core/Src/main.c) ๏ฟฝ๏ฟฝ `USER CODE BEGIN Includes` ๏ฟฝะผ๏ฟฝ๏ฟฝ๏ฟฝ `#include <stdio.h>`๏ฟฝ๏ฟฝ
4. ๏ฟฝ๏ฟฝ [Core/Src/main.c](../../Core/Src/main.c) ๏ฟฝ๏ฟฝ `USER CODE BEGIN 0` ๏ฟฝ๏ฟฝสต๏ฟฝ๏ฟฝ `_write`๏ฟฝ๏ฟฝ๏ฟฝฺฒ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `HAL_UART_Transmit(...)`๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `printf` ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝฺก๏ฟฝ
5. สน๏ฟฝ๏ฟฝ `printf` ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝอก๏ฟฝ๏ฟฝึท๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝสฑ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝศฑ๏ฟฝึค๏ฟฝ๏ฟฝสฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝฦฅ๏ฟฝไฃฌ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
6. ๏ฟฝ๏ฟฝ๏ฟฝึป๏ฟฝ๏ฟฝฦด๏ฟฝ๏ฟฝ๏ฟฝึท๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `snprintf` ๏ฟฝ๏ฟฝะด๏ฟฝ๋ปบ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `printf` ๏ฟฝ๒ดฎฟฺท๏ฟฝ๏ฟฝอก๏ฟฝ

### ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝสพ๏ฟฝ๏ฟฝ

```c
printf("Hello, world!\r\n");
printf("CH0=%ld, CH1=%ld\r\n", (long)adc0, (long)adc1);
snprintf(buf, sizeof(buf), "V=%.3fV\r\n", voltage);
printf("%s", buf);
```

### ืข๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ

1. `printf` ๏ฟฝ฿ด๏ฟฝ๏ฟฝฺป๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝวฐ๏ฟฝ฿ณฬฃ๏ฟฝฦต๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝำก๏ฟฝ๏ฟฝำฐ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝอฟ๏ฟฝ๏ฟฝ๏ฟฝสตสฑ๏ฟฝิก๏ฟฝ
2. ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝสฝ๏ฟฝ๏ฟฝอจ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝำด๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝึป๏ฟฝ๏ฟฝาช๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝสพ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝใกฃ
3. ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝัพ๏ฟฝ๏ฟฝ๏ฟฝ `_write` ๏ฟฝุถ๏ฟฝ๏ฟฝ๒ฃฌฒ๏ฟฝาช๏ฟฝุธ๏ฟฝสต๏ฟฝ๏ฟฝอฌ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
